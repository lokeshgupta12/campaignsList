# campaignsList

## Run
- Clone the project and shift to the project root folder
- npm install --save
- npm start (application will start with http://localhost:3000/)
- npm test (For test cases)
- Call AddCampaigns from Browser Console for adding campaign

## Dependencies used
- react-bootstrap
- moment
- bootstrap (v5.1.3)
- react-datepicker
- react-redux
- redux-saga
